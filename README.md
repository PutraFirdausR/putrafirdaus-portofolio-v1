## 💡 Tentang Proyek Ini

**Portfolio v1** adalah rilis pertama dari *digital resume* dan portofolio saya. Dibangun dengan fokus pada kecepatan, desain *clean UI*, dan pengalaman pengguna yang halus. Repositori ini berfungsi sebagai etalase kode dan juga tempat saya membagikan karya-karya terbaik saya kepada calon klien atau *recruiter*.

## ✨ Fitur Utama

- **Single Page Application (SPA):** Navigasi mulus tanpa *reload* halaman, didukung oleh Vue.js.
- **Desain Responsif:** Tampilan yang menyesuaikan dengan sempurna di perangkat *mobile*, *tablet*, maupun *desktop* (menggunakan Tailwind CSS).
- **Performa Cepat:** Dioptimalkan dengan *bundler* Vite untuk waktu *load* yang secepat kilat.
- **Showcase Proyek:** Menampilkan daftar proyek (seperti *Firdaus Residence*, dll) dengan *preview* interaktif.

## 🛠️ Stack Teknologi

Proyek ini dibangun menggunakan alat dan teknologi modern:

| Kategori | Teknologi |
| :--- | :--- |
| **Framework UI** | Vue.js (v3) |
| **Styling** | Tailwind CSS / PostCSS |
| **Build Tool** | Vite |
| **Deployment** | Vercel |

## 📂 Struktur Direktori

Berikut adalah gambaran singkat mengenai struktur *folder* pada proyek ini:

```text
putrafirdaus-portofolio-v1/
├── public/              # Aset publik statis (favicon, dll)
├── src/                 # Kode sumber utama (Source Code)
│   ├── assets/          # Gambar, font, css global
│   ├── components/      # Komponen Vue yang dapat digunakan ulang
│   ├── App.vue          # Root komponen Vue
│   └── main.js          # Entry point JavaScript
├── index.html           # Template utama HTML
├── tailwind.config.js   # Konfigurasi kustom Tailwind CSS
├── vite.config.js       # Konfigurasi Vite
└── package.json         # Dependensi dan script NPM
```

🚀 Instalasi & Menjalankan Lokal
Ingin melihat source code ini berjalan di mesin lokalmu? Ikuti panduan berikut:

Clone Repositori:

Bash
```bash
git clone [https://github.com/PutraFirdausR/putrafirdaus-portofolio-v1.git](https://github.com/PutraFirdausR/putrafirdaus-portofolio-v1.git)
```
Masuk ke Direktori Proyek:

Bash
```bash
cd putrafirdaus-portofolio-v1
```
Instal Dependensi NPM:

Bash
```bash
npm install
```
Jalankan Development Server:

Bash
```bash
npm run dev
```
Buka http://localhost:5173 (atau port yang tertera di terminal) pada browsermu.

Build untuk Production (Opsional):

Bash
```bash
npm run build
```
