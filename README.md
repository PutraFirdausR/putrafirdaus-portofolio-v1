## 💡 About This Project

**Portfolio v1** is the first release of my digital resume and portfolio. Built with a focus on speed, clean UI design, and a smooth user experience. This repository serves as a code showcase and a place to share my best work with potential clients or recruiters.

## ✨ Key Features

- **Single Page Application (SPA):** Seamless navigation without page reloads, powered by Vue.js.
- **Responsive Design:** Layouts that adapt perfectly across mobile, tablet, and desktop devices (using Tailwind CSS).
- **Fast Performance:** Optimized with the Vite bundler for lightning-fast load times.
- **Project Showcase:** Displays a list of projects (like *Firdaus Residence*, etc.) with interactive previews.

## 🛠️ Tech Stack

This project was built using modern tools and technologies:

| Category | Technology |
| :--- | :--- |
| **UI Framework** | Vue.js (v3) |
| **Styling** | Tailwind CSS / PostCSS |
| **Build Tool** | Vite |
| **Deployment** | Vercel |

